# U2-Elite-Foundation-Enterprise-Risk-Assessment
U2 Elite Foundation — Enterprise Cybersecurity Risk Assessment (NIST CSF Framework)
Overview

This project is a simulated enterprise risk assessment conducted for the U2 Elite Foundation, a nonprofit organization focused on youth athletics, community programs, and digital operations.
The purpose of this assessment is to identify key cyber risks, evaluate their impact and likelihood, and recommend controls aligned with the NIST Cybersecurity Framework (CSF).

Objectives

Identify critical assets

Analyze organizational threats & vulnerabilities

Rate risks using a standardized scoring model

Provide mitigation recommendations

Map controls to NIST CSF categories

Create a professional GRC case study for employment portfolios

Key Deliverables

Risk Assessment Report (PDF)

Risk Register (CSV)

Risk Heat Map

NIST CSF Mapping Table

Diagrams (System Overview)

Framework Used

NIST CSF: Identify, Protect, Detect, Respond, Recover

Risk Scoring: Likelihood x Impact (1–5 scale)

Assets Evaluated

Donor & Program Participant Data

Volunteer & Staff Personal Information

Payment Systems (Stripe/QuickBooks)

Cloud Storage (Google Workspace)

Website & Registration Portals

Social Media Accounts

Event Management Systems

Sample Risks Identified

Phishing leading to account compromise

Misconfigured cloud storage exposing sensitive data

Unpatched laptops used by staff

Weak password hygiene

Social media account takeover

Third-party payment vendor breach
